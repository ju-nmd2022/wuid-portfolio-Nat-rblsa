J2N2F

Smart Faucet

JN2F2 is a single control unit for washbasin, constructed in common materials used in faucets such as metals or plastics. It has a transparent plastic compartment, which only stores the water necessary for a certain activity, to avoid water waste. To select the required amount of water, there is a button that changes the mode of use according to the number of times it is pressed and a screen on which the selection of the user is projected.
